# Webxsid Tap

## How do I install these formulae?

`brew install webxsid/tap/<formula>`

Or `brew tap webxsid/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "webxsid/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
